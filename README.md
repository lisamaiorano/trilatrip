## Requisiti

- Node.js
- npm
- Docker e Docker Compose, se si vuole avviare PostgreSQL
- Una chiave API OpenAI

## Configurazione

Installa le dipendenze del backend:

```bash
cd backend
npm install
```

Installa le dipendenze del frontend:

```bash
cd frontend
npm install
```

Crea un file `.env` dentro la cartella `backend`:

```env
OPENAI_API_KEY=la_tua_chiave_openai
```

## Avvio del progetto

Avvia il backend:

```bash
cd backend
npm run dev
```

Il backend risponde su:

```text
http://localhost:3001
```

Avvia il frontend in un secondo terminale:

```bash
cd frontend
npm run dev
```

Il frontend viene servito da Vite, di solito su:

```text
http://localhost:5173
```

## Avvio del database

Per avviare PostgreSQL con Docker:

```bash
docker compose up -d
```

Configurazione database:

```text
Host: localhost
```

## Flusso utente

1. L'utente arriva sulla Home.
2. Clicca su "Start planning" e apre la pagina per creare un viaggio.
3. Compila destinazione, giorni, budget e interessi.
4. Clicca su "Generate my trip".
5. Il frontend invia i dati al backend.
6. Il backend genera l'itinerario con OpenAI.
7. L'utente visualizza il risultato nella pagina dedicata.
